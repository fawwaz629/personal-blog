# 📝 Personal Blog — Muhammad Fawwaz bin Kamarul Shukri

A responsive personal blog and portfolio website built with pure HTML, CSS, and JavaScript as part of the **CSD 34203 Special Topics in Software Development** course at Universiti Sultan Zainal Abidin (UniSZA).

---

## 🌐 Demo

> Deploy using GitHub Pages and paste your live link here  
> `https://fawwazks.github.io/personal-blog/`

---

## ✨ Features

- **4 Pages** — Home, About, Blog, Contact (Single-Page Application)
- **Typing Animation** — Animated role text on the hero section
- **Blog Filter** — Filter posts by category (Web Dev, Python, Git, Tips)
- **Animated Skill Bars** — Progress bars animate when the About page loads
- **Contact Form** — With success state feedback
- **Responsive Design** — Mobile-friendly with hamburger menu
- **Dark Theme** — Navy + teal + amber colour palette
- **Smooth Navigation** — Page transitions with fade animation

## 🚀 How to Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/fawwazks/personal-blog.git
   ```

2. **Open the project**
   ```bash
   cd personal-blog
   ```

3. **Launch in browser**
   - Open `index.html` directly in your browser, **or**
   - Use VS Code **Live Server** extension for hot reload

> No build tools or npm required — runs straight in the browser!
